# PureSoulJoy

AI-powered lifestyle health system focused on building consistent daily habits for long-term health and inner well-being.

## 🚀 Features

- Chat-based health insights
- Meal and lifestyle guidance
- Glucose, BP, and cholesterol awareness
- Personalized feedback based on user input

## 🧠 Vision

PureSoulJoy aims to combine:
- Data-driven health insights
- Behavioral consistency
- Inner well-being (soul-level alignment)

## 🛠 Tech Stack

- React Native (Expo)
- Python backend (FastAPI)
- OpenAI API
